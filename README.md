# Mrs. Rogers’ Homebase Arcade

Static GitHub Pages classroom games. No build step. No external assets required.

Root files:
- `index.html` — hub
- `portalpark.html` — Portal Park
- `pokemonmath.html` — Pokémon Math Quest
- `wordquest.html` — Word Quest

Shared local files:
- `css/arcade.css`
- `js/arcade.js`
- `assets/` for future local art/audio

Live URL after GitHub Pages deployment:
