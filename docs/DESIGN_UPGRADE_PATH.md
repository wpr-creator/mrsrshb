# Best design upgrade path

This repaired package keeps the current HTML games working and adds a shared offline polish layer.

Next quality jump:

1. Rebuild `portalpark.html` first as the prototype.
2. Use Canvas or Phaser for animated maps, characters, rewards, and transitions.
3. Keep actual questions and answer buttons in readable HTML for second-grade Chromebook use.
4. Add local CC0 assets into `assets/` only after license checking.

Recommended free sources to verify before use:
- Phaser 3 for Canvas/WebGL game structure.
- Kenney CC0/public-domain asset packs for sprites, UI, and tiles.
- itch.io CC0 packs only when the license page clearly says CC0/public domain.

Do not link to CDNs if offline Chromebook use is required. Put every file in this repo.
