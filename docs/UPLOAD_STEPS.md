# Upload steps

1. Open your cloned `mrsrshb` repo folder in VS Code.
2. Drag the contents of this package into the repo root.
3. Replace existing files when asked.
4. In the VS Code terminal, run:

```bash
git add .
git commit -m "Repair arcade package"
git push
```

5. Wait about one minute, then hard-refresh the GitHub Pages site.

Quick check: View Source on the live homepage and search for `css/arcade.css`.
